# CLI Data Pipeline

Starter repository for the **Python for Engineers** capstone on [DutchEngineer](https://dutchengineer.org).

## What you're building

A command-line pipeline that reads, transforms, and validates a dataset using idiomatic Python. The pipeline must be typed, tested, and runnable by another engineer without your help — proving you can write Python that holds up under real usage, not just in a notebook.

## Getting started

1. Fork this repository
2. Clone your fork: `git clone https://github.com/<your-username>/cli-pipeline-starter`
3. Follow the capstone instructions in your DutchEngineer dashboard
